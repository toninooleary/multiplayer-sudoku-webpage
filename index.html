Multiplayer Sudoku

The logic-based game Sudoku is a slow paced puzzle game where players place numbers on a grid. This webpage explores multiplayer variations with physical demos of proposed game ideas. These aim to change the dynamics of the traditional game by attempting to create a fun fast paced experience.

Game Rules

Figure 1 rules

  1. Fill and complete grid before opponent
  2. Grid rows and columns can only contain 1 instance of each number
  3. When a player beats opponent, move player's position up 'step indicator'
  4. Once player reaches their final goal on 'step indicator', player wins

Figure 2 rules

  1. Form a line of six squares containing all of the six numbers
  2. Lines can be formed vertically and horizontally
  3. Any row filled will collapse
  4. The player who completes the line gain the point
  5. Game ends when no more empty sqaures remain
  6. Player with the most points wins
